# Requirements

## Functional Requirements

**RF01** – The system shall allow users to register subjects.

**RF02** – The system shall allow users to edit subjects.

**RF03** – The system shall allow users to delete subjects.

**RF04** – The system shall allow users to register tasks.

**RF05** – The system shall allow users to edit tasks.

**RF06** – The system shall allow users to delete tasks.

**RF07** – The system shall allow users to mark a task as completed.

**RF08** – The system shall allow users to filter tasks by subject and status.

**RF09** – The system shall automatically calculate the user's study progress.

---

## Non-Functional Requirements

**RNF01** – The system shall store user data using Local Storage.

**RNF02** – The system shall provide a responsive user interface.

**RNF03** – The system shall be compatible with modern web browsers.

**RNF04** – The system shall provide a simple and intuitive user experience.
