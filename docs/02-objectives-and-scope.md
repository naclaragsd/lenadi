# Objectives and Scope

## General Objective

Develop a system to help students organize and manage their academic routine.

## Specific Objectives

* Manage academic subjects.
* Manage study tasks.
* Track academic progress.
* Facilitate study planning.
* Store user data automatically.

## Scope

The first version of Lenadi includes the following features:

* Subject registration.
* Subject editing.
* Subject deletion.
* Task registration.
* Task editing.
* View task details.
* Task deletion.
* Mark tasks as completed.
* Filter tasks by subject.
* Automatic study progress calculation.
* Automatic data storage using Local Storage.

## Out of Scope

The following features are not included in Version 1 and are planned for future releases:

* Artificial Intelligence (AI)
* Electron desktop application
* SQLite database
* Flashcards
* PDF management
* User authentication (Login)
* Notifications
* Study summaries
* Study recommendations
