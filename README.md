
# Lenadi

Lenadi is a study management system designed to help students organize subjects, tasks, study materials and academic progress.

## Status

🚧 In development

## Technologies

- HTML5
- CSS3
- JavaScript
- localStorage

## Version 1 features

- Subject registration
- Task registration
- Edit and delete subjects
- Edit and delete tasks
- Mark tasks as completed
- Filter tasks by subject
- Progress calculation
- Automatic data saving with localStorage
